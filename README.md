# Нашествие Они

Top-down 2D JRPG-style игра с элементами bullet-hell разработанная на open-source движке [Godot 4.6](https://github.com/godotengine)
под разрешительной лицензией [MIT](LICENSE) студентами 2-ого курса [Самарского национального исследовательского университета имени академика С.П. Королева](https://ssau.ru).

В качестве основы проекта используется ассет-пак [Ninja Adventure](https://pixel-boy.itch.io/ninja-adventure-asset-pack) с [itch.io](https://itch.io) от [Pixel-Boy](https://twitter.com/2Pblog1) и [AAA](https://www.instagram.com/challenger.aaa/?hl=fr).

## Играть

В данный момент ранний билд игры доступен в [разделе релизов](https://github.com/barinovadn/umazing-game/releases) полностью на русском языке для windows.

### [⬇️ Скачать и играть бесплатно полностью на русском языке без регистрации 🔥](https://github.com/barinovadn/umazing-game/releases)
